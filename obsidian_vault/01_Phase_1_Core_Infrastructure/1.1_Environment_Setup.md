# 1.1 Environment Setup
Status: - [ ] Not Started

## Description
Set up the foundational project configuration: environment variables, Python dependencies, and a centralized config module that loads settings from `.env`.

## Files to Create/Edit
- [ ] `.env`
- [ ] `requirements.txt`
- [ ] `src/config.py`

```

Perform the following:
```
